# [Solutioning] Warvote UI/UX 화면 기획 및 사용자 여정

## 1. 페르소나별 주요 기능(Screen) 구성

### A. 활동가/조직원 (Mobile-First Web)
1.  **내 대시보드:** 내가 등록한 인원수, 내 지인들의 평균 메시지 반응률(클릭률), 캠페인 목표 달성도.
2.  **지인 등록/수정:** 스마트폰 주소록 연동 또는 수동 입력을 통한 유권자 프로파일링 입력 화면.
3.  **메시지 전파:** 관리자가 배포한 캠페인 메시지를 확인하고 지인들에게 재전파(카톡/문자)하는 버튼.

### B. 관리자/전략가 (Desktop Dashboard)
1.  **워게임 커맨드 센터:** 실시간 이슈 알림, 주요 지표(총 지지자 수, 오늘 발생한 주요 영상/기사).
2.  **관계형 CRM 관리:** 전체 활동가 현황, 타겟팅 메시지 발송기, 클릭 트래킹 보고서.
3.  **이슈 추적 맵:** 특정 이슈의 커뮤니티 확산 경로 시각화.
4.  **전략 맵 (GIS):** 지도 위에 유권자 밀집도와 후보자 동선 시각화 (Phase 3 연동).

## 2. 핵심 사용자 여정 (User Journey)

### [CRM: 지인 등록 및 메시지 반응 추적]
1.  **활동가**가 시스템에 접속하여 '지인 등록' 선택.
2.  이름, 지역, 관심사를 입력하면 **시스템**에서 고유 유권자 ID 부여.
3.  **관리자**가 특정 '유튜브 링크'를 포함한 문자 캠페인 실행.
4.  **동네 지인**이 문자를 받고 링크 클릭 -> **시스템**에 '서울 강남구 40대 남성 클릭' 로그 저장.
5.  **관리자** 화면에 실시간 클릭 카운트 및 타겟팅 적중률 업데이트.

### [인텔리전스: 부정 정보 대응]
1.  **시스템**이 유튜브에서 후보자 비방 키워드가 포함된 영상 포착.
2.  **전략가** 대시보드에 '긴급 알림' 노출 및 해당 영상의 확산 예측(조회수 급증 등) 표시.
3.  **전략가**가 대응 콘텐츠(카드뉴스/반박영상)를 **활동가**들에게 전파 요청.
4.  **활동가**들이 각자 지인들에게 해당 콘텐츠 발송.

## 3. 화면 와이어프레임 (주요 기능)

*   **[대시보드 상단]:** 실시간 여론 온도계 (Positive/Neutral/Negative).
*   **[중앙 맵]:** 이슈 이동 경로 (Node-Link Diagram) 또는 GIS 전략 맵.
*   **[우측 패널]:** 긴급 모니터링 피드 (최신 기사 및 커뮤니티 베스트 글).
*   **[하단 통계]:** 메시지 발송 현황 및 전환율(CTR).
